# Data Science Tools and Ecosystem

## Project Overview
This repository contains the final project for the "Tools for Data Science" course by IBM on Coursera. The project demonstrates a fundamental understanding of the data science ecosystem, including programming languages, software libraries, and development environments commonly utilized by data scientists.

## Notebook Content
The primary work is contained within the `DataScienceEcosystem.ipynb` notebook. It includes:
* **Languages for Data Science:** A list of popular languages used in the field.
* **Data Science Libraries:** Commonly used open-source libraries for data manipulation, visualization, and machine learning.
* **Development Tools:** A structured table summarizing key development environments (IDEs).
* **Arithmetic Expressions:** Simple Python code cell examples illustrating basic arithmetic evaluations and time conversions (minutes to hours).

## Tools Used
* **Environment:** JupyterLab / IBM Skills Network Labs
* **Language:** Python 3.8
